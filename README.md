# 🤖 Nexus AI Analytics Platform

AI-Powered Business Intelligence & Data Analytics System built using Python, Streamlit, Groq, Plotly, and Machine Learning.

---

# 🚀 Overview

Nexus AI Analytics Platform is an advanced AI-driven business intelligence application that allows users to upload CSV datasets, generate interactive visualizations, detect anomalies using machine learning, and receive AI-powered business insights through natural language interaction.

The platform is designed to simulate a modern SaaS analytics product with a professional dashboard, AI assistant, anomaly detection engine, and intelligent visual analytics.

# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/nexus-ai-analytics.git
```

---

## Navigate to Project

```bash
cd nexus-ai-analytics
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# 🔑 Environment Variables

Create a `.env` file and add:

```env
GROQ_API_KEY=your_api_key_here
```

---

# ▶️ Run Application

```bash
streamlit run data_agentic.py
```
